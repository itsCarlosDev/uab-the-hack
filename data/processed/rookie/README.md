# Datos rookie procesados (solo local)

Esta carpeta debe almacenar `rookie_filtered_aps.json` (~1 GB) y `rookie_filtered_clients.json` (~3 GB). Ambos salen de `docs/hackathon-kit/scripts/create_filtered_json.py` a partir de `data/raw/anonymized_data`.

GitHub no acepta archivos de este tamano (>100 MB), asi que mantenlos fuera del control de versiones y compartelos mediante un almacenamiento externo cuando el equipo los necesite.
